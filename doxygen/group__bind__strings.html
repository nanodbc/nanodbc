Detailed Description

If your prepared SQL query has any ? placeholders, this is how you bind values to them. Placeholder numbers count from left to right and are 0-indexed.

It is possible to use these functions for bulk operations.

Parameters
paramPlaceholder position.
valuesValues to substitute into placeholder.
lengthMaximum length of string elements.
elementsThe number of elements being bound. Otherwise the value N is taken as the number of elements.
null_sentryValue which should represent a null value.
param_direcitonODBC parameter direction.
Exceptions
database_error
